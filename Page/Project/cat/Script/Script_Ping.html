Script Ping

Questo script permette di eseguire il ping per n volte e salvare in un file oppure no.

1 #!/bin/bash: Shebang che indica l'uso del programma Bash per eseguire lo script.
2 echo "Numero esecuzioni: ": Stampa a schermo il messaggio "Numero esecuzioni: ".
3 read n: Legge l'input dell'utente e assegna il valore inserito alla variabile n.
4 echo "IP: ": Stampa a schermo il messaggio "IP: ".
5 read IP: Legge l'input dell'utente e assegna il valore inserito alla variabile IP.
6 echo "Output [1] file o [2] line: ": Stampa a schermo il messaggio "Output [1] file o [2] line: ".
7 read comm: Legge l'input dell'utente e assegna il valore inserito alla variabile comm.
8 if [[ "$comm" = "1" ]]; then: Inizia una struttura di controllo if che verifica se il valore di comm è uguale a "1".
9 echo "Nome file": Stampa a schermo il messaggio "Nome file".
10 read nomefile: Legge l'input dell'utente e assegna il valore inserito alla variabile nomefile.
11 if [ -e "$nomefile" ]; then: Verifica se il file specificato da nomefile esiste.
12 echo "Il file esiste!": Stampa a schermo il messaggio "Il file esiste!".
13 echo "[1] sovrascrivere o [2] appendere": Stampa a schermo il messaggio "[1] sovrascrivere o [2] appendere".
14 read rix: Legge l'input dell'utente e assegna il valore inserito alla variabile rix.
15 if [[ "$rix" = "1" ]]; then: Inizia una struttura di controllo if che verifica se il valore di rix è uguale a "1".
16 ping -w $n $IP > $nomefile: Esegue il comando di ping con l'opzione -w specificata da n e l'indirizzo IP specificato da IP. L'output del comando di ping viene redirezionato nel file specificato da nomefile, sovrascrivendo eventuali contenuti esistenti.
17 elif [[ "$rix" = "2" ]]; then: Inizia una struttura di controllo elif che verifica se il valore di rix è uguale a "2".
18 ping -w $n $IP >> $nomefile: Esegue il comando di ping con l'opzione -w specificata da n e l'indirizzo IP specificato da IP. L'output del comando di ping viene aggiunto al file specificato da nomefile, senza sovrascriverne il contenuto.
19 elif [[ "$comm" = "2" ]]; then: Inizia una struttura di controllo elif che verifica se il valore di comm è uguale a "2".
20 ping -w $n $IP: Esegue il comando di ping con l'opzione -w specificata da n e l'indirizzo IP specificato da IP, senza alcuna redirezione dell'output.
21 else: Blocco eseguito se nessuna delle condizioni precedenti è vera.
22 echo "errore": Stampa a schermo il messaggio "errore".
