Script SH

Uno script sh è un file di testo contenente una serie di comandi scritti nel linguaggio di scripting della shell, come Bash. Questi comandi possono essere eseguiti sequenzialmente dalla shell di un sistema operativo Unix o Unix-like. Gli script sh sono utilizzati per automatizzare compiti, creare programmi di installazione, avviare servizi e altro ancora. Possono essere eseguiti dalla riga di comando e richiedono i permessi di esecuzione per essere chiamati correttamente.

Script Personale
SCRIPT LEGGI

Leggere un File riga per riga senza doverlo aprire.

Script Personale
SCRIPT ELIMINA

Chiede all'utente il nome del file da eliminare ed esegue "rm" (è facilmente modificabile per dargli direttamente alla chiamata il nome del file).

Script Personale
SCRIPT ATTESA

Per attendere il completamento di un qualsiasi processo in esecuzione.

Script Personale
SCRIPT DIRECTORY

Chiede all'utente il nome con cui si vuole nominare la cartella ed esegue "mkdir" (è facilmente modificabile per dargli direttamente alla chiamata il nome del file).

Script Personale
SCRIPT SOMMA

Quando viene richiamato il file si passano 2 argomenti (X e Y) lo script eseguirà la somma dei due.

Script Scuola
SCRIPT PING

Esegue il ping su un indirizzo scelto e dato dall'utente, si può decidere se inserire il risultato su bash o all'interno di un file sovrascrivendolo oppure no.
